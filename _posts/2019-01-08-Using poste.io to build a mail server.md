# start poste.io docker
```
docker run \
-p 25:25 \
-p 80:80 \
-p 110:110 \
-p 143:143 \
-p 443:443 \
-p 587:587 \
-p 993:993 \
-p 995:995 \
-v /etc/localtime:/etc/localtime:ro \
-v /var/maildata:/data \
--name "mail" \
-h "mail.dba.pw" \
-t analogic/poste.io
```

username: line@dba.pw
